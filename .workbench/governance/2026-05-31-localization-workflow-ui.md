# Claude Desktop 汉化流程页

## 变更原因
原汉化页右侧状态区重复展示当前语言、检测状态、备份状态和资源路径，信息密度低；用户要求按健康检查式流程重排，并默认只做简体中文汉化。

## 本轮变更
- 前端汉化页改为六行流程：汉化程度、备份状态、汉化范围、一键汉化、重启 Claude Desktop、恢复英文。
- 移除语言选择，安装命令固定传入 `zh-CN`。
- 新增独立备份命令，供“无备份时显示备份按钮”调用；该命令只创建恢复备份，不安装汉化、不改语言。

## 验证
- 已用内容检查确认页面不再包含语言下拉，并包含新的流程布局。
- 后续需运行前端构建和浏览器页面验收。

## 回退依据
回退本轮提交即可恢复原页面；Claude Desktop 资源恢复仍使用应用内“恢复英文”动作和既有备份目录。
